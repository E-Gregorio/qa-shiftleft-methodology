🔗 Integración de las guías

Las guías están diseñadas para usarse de forma complementaria:

  • VCR → método de estimación y priorización por valor/costo/riesgo
  • Estrategia de Pruebas → cómo ejecutar las pruebas eficientemente
  • Gestión de Riesgos → marco para tratar riesgos identificados
  • Testing IA/LLM → extensión para sistemas con componentes IA
  • Performance → extensión para validar tiempos y capacidad
  • DevSecOps → extensión transversal de seguridad en cada etapa
